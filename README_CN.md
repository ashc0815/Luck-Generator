# 🍀 Luck Generator — 运气制造机

一个 [OpenClaw](https://github.com/openclaw/openclaw) 技能，让你的 AI agent 变成一台每日运气制造机。

> **运气不是随机的，它可以被系统性地制造。**

## 为什么做这个

这个 skill 诞生于对"运气的科学"的研究阅读。三项研究改变了一切：

1. **感恩日记** — Emmons & McCullough（2003，*Journal of Personality and Social Psychology*）的里程碑研究证明：每天记录感恩事项的参与者幸福感提升25%，运动量更大，而且——关键的是——更愿意主动帮助他人。感恩不只是改善心情，它重新训练了你的注意力系统。

2. **日行一善** — Lyubomirsky 等人（2005，*Review of General Psychology*）的干预实验表明：每周做5件善事能显著提升幸福感。每一次善行都在扩大你的"运气表面积"（Jason Roberts / Sahil Bloom 的概念）——你正向影响的人越多，回流的"好运"连接就越多。

3. **跨界学习** — Wiseman（2003，*The Luck Factor*）用10年研究了400+名"幸运"和"不幸"的人。他最著名的报纸实验：他在报纸中放了一个大通知"告诉实验者你看到了这条消息，赢得250美元"。"不幸"的人全部错过了——因为他们太专注于数照片。"幸运"的人立刻就看到了。**运气是一个注意力问题——而这三个习惯就是在训练你的注意力。**

**运气 = 善行 × 觉察力 × 好奇心，每日复利。**

## 工作原理

| 支柱 | 做什么 | 什么时候 |
|------|--------|---------|
| 🤝 日行一善 | 记录一件善行（扩大运气表面积） | 晚上 |
| 🙏 感恩日记 | 记录一件感恩的事（磨练觉察力） | 晚上 |
| 🧠 跨界学习 | 学一个专业之外的知识点（三层领域引擎） | 早上 |

AI agent 会主动找你互动——早上推送跨界知识，晚上引导反思签到——并将所有数据综合成一份**运气周报**，包含 **运气评分（0-100）**、模式分析、和下周具体行动路径。

### 三层领域引擎

| 层级 | 占比 | 目的 | 示例 |
|------|------|------|------|
| 🎣 惊喜钩子 | 60% | 每日多巴胺，适合分享 | 心理学实验、食物科学、动物行为、博弈论 |
| 🔬 硬核知识 | 20% | 长期构建世界观 | 物理、天文、历史转折点、生物进化、芯片半导体 |
| 🔗 隐性关联 | 20% | 桥接你的专业领域 | 供应链↔数据管道、航空安全↔合规规则、AI时事的跨域视角 |

## 安装

```bash
# 方式1：从 ClawHub 安装
clawhub install luck-generator

# 方式2：从本仓库安装
# 把本仓库 URL 发给你的 OpenClaw，它会自动处理
```

或者手动复制 `SKILL.md` 和 `references/` 文件夹到：
```
~/.openclaw/skills/luck-generator/
```

## 使用方法

| 命令 | 功能 |
|------|------|
| `运气制造机` / `luck generator` | 今日状态面板 |
| `今日学习` / `teach me something` | 获取一条跨界知识 |
| `晚间签到` / `evening check-in` | 善行 + 感恩记录 |
| `周报` / `weekly report` | 生成运气周报 |
| `回顾` / `review` | 7天回顾 + 模式分析 |

## 自动化（推荐）

设置 cron 定时任务，获得完整的主动式体验：

```bash
# 早间学习推送（早上8点）
0 8 * * * openclaw agent --message "/luck-generator 今日学习"

# 晚间签到（晚上9点）
0 21 * * * openclaw agent --message "/luck-generator 晚间签到"

# 运气周报（周日早上10点）
0 10 * * 0 openclaw agent --message "/luck-generator 周报"
```
